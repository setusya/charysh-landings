Сайт состоит из трёх React-страниц на статическом стеке (React 18 + Babel
Standalone, без сборки):

| Файл | Назначение |
|------|------------|
| `privatecamps.html` | Основной инвест-лендинг для Private Camp 2 (~30 секций) |
| `8-lines.html` | Лендинг для сообщества Архстояние / студии 8 Lines (17 слайдов): архитектура + recovery + три формы инвестиций |
| `familycamp.html` | Лендинг детско-родительского кэмпа |
| `Лендинг для сбора контактов.html` | Короткая форма-визитка |

## Как открыть локально

React + Babel компилируются в браузере, поэтому нужен локальный HTTP-сервер
(через `file://` JSX не загрузится).

### Python
```bash
python -m http.server 8000
# http://localhost:8000/privatecamps.html
```

### Node
```bash
npx live-server --port=5500
```

### VS Code
Установить расширение **Live Server** → ПКМ по `privatecamps.html` →
*Open with Live Server*.
